# Sleeptrackwebapp

A web application for sleep intervention and health tracking.

## Prerequisites

- Node.js 18+
- npm

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Notes

Create a `.env` file if needed and add your Supabase variables:

- `VITE_SUPABASE_URL` (or `VITE_SUPABASE_PROJECT_ID`)
- `VITE_SUPABASE_ANON_KEY`
